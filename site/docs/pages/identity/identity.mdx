# `<Identity />`

`Identity` is a React context provider and arranges identity components.

## Usage

Show user avatar and address:

```tsx twoslash
import { Avatar, Identity, Name, Address } from '@coinbase/onchainkit/identity';

<Identity // [!code focus]
  address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9" // [!code focus]
> // [!code focus]
  <Avatar /> // [!code focus]
  <Name /> // [!code focus]
  <Address /> // [!code focus]
</Identity> // [!code focus]
```

<App>
  <Identity
    address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
  >
    <Avatar />
    <Name />
    <Address />
  </Identity>
</App>


Modify any styles with `className` prop.

```tsx twoslash
import { Avatar, Identity, Name, Address } from '@coinbase/onchainkit/identity';

<Identity
  address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
>
  <Avatar />
  <Name className="text-orange-600" /> // [!code focus]
  <Address className="text-gray-500 font-bold"/> // [!code focus]
</Identity>
```

<App>
  <Identity
    address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
  >
    <Avatar />
    <Name className="text-orange-600" />
    <Address className="text-gray-500 font-bold"/>
  </Identity>
</App>

Choose which identity components to render:

```tsx twoslash
import { Avatar, Identity, Name, Address } from '@coinbase/onchainkit/identity';
// ---cut-start---
<div>
// ---cut-end---

<Identity address="0x838...">
  <Avatar /> // [!code focus]
  <Name /> // [!code focus]
</Identity>

<Identity address="0x838...">
  <Name /> // [!code focus]
  <Address /> // [!code focus]
</Identity>
// ---cut-after---
</div>
```

<App>
  <Identity
    address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
  >
    <Avatar />
    <Name />
  </Identity>
  <Identity
    address="0x838aD0EAE54F99F1926dA7C3b6bFbF617389B4D9"
    className="mt-2"
  >
    <Name />
    <Address />
  </Identity>
</App>
